# AI 행동 지침 (User Preferences)

1. **언어**: 모든 대답과 주석은 **한국어**로 작성한다.
2. **유지**: 이 파일은 프로젝트 루트에 위치하며, AI가 매 세션마다 참고하여 사용자의 선호를 유지한다.
3. **테스트**: 모든 기능들은 추가될때 테스트 코드를 작성하고 feature.md에 해당기능을 추가한다. 기능작성이 완료되면 유닛테스트를 실행하고 문제가 없는지 확인한다.
4. **컨텐츠 생성**: [퀴즈 생성 지침](gemini/quiz-generate.md) 및 [통합 이미지 생성 프로토콜](gemini/image-production-protocol.md)을 준수한다.
5. **이미지 생성 및 업로드**:
    - 모든 이미지는 webp로 변환하여 supabase에 업로드한다. (사이즈 선정은 AI 재량에 맡김)
    - 이미지 URL에는 캐시 버스팅을 위해 `?v=타임스탬프` 파라미터를 추가한다.
    - **쿼터 제한 발생 시**: 즉시 `scripts/generate-nanobanana-assets.ts --use-ai-studio`를 사용하여 **Gemini 2.5 Flash** 모델로 생성한다.
6. **작업 관리**: 각각의 작업은 `work` 폴더 내에 `.md` 파일로 작성하며, 작업 내용(수행한 항목 및 미수행 항목)을 반드시 기록한다.
7. **데이터 삭제금지**: 절대로 기존에 있던 Supabase DB, 파일스토리지는 삭제하지마. 삭제해야될 일이 있으면 항상 나에게 물어봐
