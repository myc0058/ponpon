# SEO 및 메타 태그 작업 리스트

## 1. Google SEO (검색 엔진 최적화)
*   **`<title>` 태그**: 각 페이지별 고유하고 설명적인 제목 작성 (키워드 포함).
*   **Meta Description**: `<meta name="description" content="...">` 태그로 페이지 요약 제공 (클릭률에 영향).
*   **Canonical URL**: `<link rel="canonical" href="...">` 태그로 중복 URL 방지 및 선호 URL 지정.
*   **H1 태그 사용**: 각 페이지(특히 퀴즈 제목)에 `<h1>` 태그를 하나씩만 명확하게 사용.
*   **Sitemap.xml**: 검색 엔진에 사이트 구조 제출.
*   **Robots.txt**: 크롤러 접근 범위 설정.
*   **JSON-LD (Structured Data)**: 콘텐츠(퀴즈 등)에 대한 구조화된 데이터를 제공하여 검색 결과 품질 향상.

## 2. Open Graph (OG) - 페이스북, 카카오톡 등
*   **og:title**: 소셜 미디어 공유 시 표시될 제목.
*   **og:description**: 공유 시 표시될 설명.
*   **og:image**: 썸네일 이미지 (권장 해상도 1200x630px).
*   **og:url**: 콘텐츠의 정규 URL.
*   **og:type**: 콘텐츠 유형 (`website`, `article` 등).
*   **og:site_name**: 사이트 이름.

## 3. Twitter Cards - 트위터
*   **twitter:card**: 카드 타입 (`summary`, `summary_large_image` 등). 이미지를 강조하려면 `summary_large_image` 추천.
*   **twitter:title**: 트위터 카드의 제목.
*   **twitter:description**: 트위터 카드의 설명.
*   **twitter:image**: 트위터 카드용 이미지.
