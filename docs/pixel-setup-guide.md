# 메타 비즈니스 설정 및 픽셀 ID 발급 가이드

메타 광고를 시작하기 위해 가장 먼저 필요한 과정입니다. 아래 단계를 따라 픽셀 ID를 생성해 주세요.

## 1. 메타 비즈니스 설정
1. [메타 비즈니스 설정](https://business.facebook.com/settings) 페이지로 이동합니다.
2. 비즈니스 계정이 없다면 먼저 생성해야 합니다.

## 2. 데이터 소스 (픽셀) 생성
1. 왼쪽 메뉴에서 **[데이터 소스]** > **[데이터 세트]**(또는 픽셀)를 클릭합니다.
2. **[추가]** 버튼을 클릭합니다.
3. 픽셀 이름(예: PonPon Web Pixel)을 입력하고 **[만들기]**를 클릭합니다.

## 3. 픽셀 ID 확인 및 복사
1. 생성된 데이터 세트 화면 상단 또는 상세 정보에서 **ID: 123456789012345** 형식의 숫자를 찾을 수 있습니다.
2. 이 숫자가 바로 **메타 픽셀 ID**입니다.

## 4. 사이트 연동 (코드 수정)
1. 복사한 ID를 제게 알려주시거나, 직접 `.env` 파일에 추가해 주시면 됩니다.
2. `.env` 파일에 `NEXT_PUBLIC_FACEBOOK_PIXEL_ID=자기의_ID` 형태로 저장하면 제가 작성한 코드가 자동으로 작동합니다.

> [!TIP]
> 픽셀이 정상적으로 설치되었는지 확인하려면 크롬 확장 프로그램인 **[Meta Pixel Helper]**를 설치하여 사용해 보세요. 사이트 접속 시 픽셀 로딩 여부를 실시간으로 알려줍니다.
