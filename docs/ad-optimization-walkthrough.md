# 메타 픽셀 연동 및 이벤트 추적 구현 완료

메타 광고를 위한 기본적인 데이터 추적 인프라 구축을 완료했습니다. 이제 픽셀 ID만 설정하면 메타 광고를 시작할 수 있는 준비가 끝납니다.

## 구현 내용

### 1. 메타 픽셀 컴포넌트 (`MetaPixel.tsx`)
- 메타 픽셀 베이스 코드를 포함하는 전용 컴포넌트를 생성했습니다.
- 환경 변수(`NEXT_PUBLIC_FACEBOOK_PIXEL_ID`)를 통해 픽셀 ID를 관리할 수 있도록 설계했습니다.
- 클라이언트 사이드 네비게이션 시에도 페이지 뷰를 정확히 추적합니다.

### 2. 주요 이벤트 추적 연동
단순히 방문자만 보는 것이 아니라, 광고의 '품질'을 측정하기 위해 다음 이벤트를 심었습니다.
- **테스트 시작 (`Lead`)**: 사용자가 퀴즈의 [시작하기]를 눌러 플레이 페이지에 진입할 때 발생합니다.
- **결과 보기 (`ViewContent`)**: 테스트를 마치고 결과 페이지를 볼 때 발생합니다.
- **결제 시도 (`InitiateCheckout`)**: 프리미엄 결과 확인을 위해 결제 버튼을 누를 때 발생합니다.
- **결제 완료 (`Purchase`)**: (현재 데모 상태) 결제가 성공적으로 완료되었을 때 발생합니다.

### 3. 종합 광고 레이아웃 및 전략 구현
vonvon 등 대형 심리테스트 사이트의 전략을 벤치마킹하여 광고 수익을 극대화할 수 있는 레이아웃을 구현했습니다.

- **데스크탑 사이드바 광고**: 1200px 이상 해상도에서 화면 좌우 여백에 **160x600** 사이즈 광고를 고정 노출합니다.
- **플레이 페이지**: 
    - 답안 선택지 바로 아래에 **300x250** 구글 광고를 배치했습니다.
    - 답안 클릭 시 **30% 확률**로 전면 팝업 광고(구글 또는 쿠팡)가 노출되어 클릭을 유도합니다.
- **로딩(Analyzing) 페이지**:
    - 상단 구글 광고 + 중앙 분석바 + [쿠팡] - [결과 버튼] - [구글] 순의 **샌드위치 배치**로 버튼 클릭 전후 광고 노출을 극대화했습니다.
- **결과 이미지 페이지**: 
    - 결과 내용과 하단 '추천 퀴즈' 리스트 사이에 구글 광고를 배치하여 자연스러운 흐름 속에 광고가 위치하도록 했습니다.

### 광고 사이즈 추천 가이드
| 위치 | 권장 사이즈 | 비고 |
| :--- | :--- | :--- |
| 사이드바 (데스크탑) | 160x600 | 좌우 여백 활용 |
| 페이지 내부 (인라인) | 300x250, 320x100 | 모바일 최적화 |
| 전면 팝업 | 300x250 | 사용자 시선 집중 |

## 다음 단계

사용자께서는 다음 작업을 수행해 주시면 됩니다.
1. [메타 픽셀 가이드](file:///home/myc0058/.gemini/antigravity/brain/be02db92-1b58-4043-944a-2369da1ad2e6/pixel_setup_guide.md)를 참고하여 픽셀 ID를 발급받으세요.
2. 프로젝트 루트의 `.env` 파일에 아래 줄을 추가해 주세요:
   ```env
   NEXT_PUBLIC_FACEBOOK_PIXEL_ID=자기의_픽셀_ID
   ```
3. 각 광고 위치(`QUIZ_PLAY_BOTTOM_SLOT`, `ANALYZING_TOP_SLOT` 등)에 애드센스 대시보드에서 생성한 실제 **Slot ID**를 넣어주시면 광고가 실제 송출됩니다.
4. 크롬에서 `Meta Pixel Helper` 확장을 설치하여 이벤트 작동 여부를 최종 확인하세요!

작업이 완료되면 메타 광고 관리자(Ads Manager)에서 '픽셀' 또는 '데이터 세트' 항목을 통해 데이터가 들어오는 것을 확인하실 수 있습니다.
